# PMP-IPTV
#### `P`inaka `M`alupit na `P`inoy IPTV playlist

Local Pinoy IPTV playlist for Filipino fellas out there.

## How to use
Add this link to your IPTV player (VLC, TiviMate, Kodi, etc.):
```
https://mxggot.github.io/PMP-IPTV/ph.m3u
```
